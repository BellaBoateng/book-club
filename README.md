# 📚 Book Club — Digital Library

A curated digital library designed to explore storytelling through **design, interaction, and code**.

This project is a front-end web experience where books aren’t just displayed — they’re **organized, explored, and felt** through thoughtful UI/UX decisions.

---

## ✨ Concept

This site was built as a personal space to:

* Document books I’ve read and connected with
* Explore how design can shape the way we experience stories
* Combine **creativity + code** into something interactive and intentional

It’s not just a bookshelf — it’s a **designed experience of reading culture**.

---

## 🖥️ Features

* 📚 Genre-based filtering (Romance, Manga, Literary, etc.)
* 🔍 Dynamic book rendering using JavaScript
* 📖 Individual book detail pages
* 👤 Author integration with visuals
* 🎨 Custom UI design with a soft, editorial aesthetic
* 📱 Responsive layout for different screen sizes

---

## 🛠️ Built With

* HTML
* CSS
* JavaScript
* GitHub Pages (deployment)
